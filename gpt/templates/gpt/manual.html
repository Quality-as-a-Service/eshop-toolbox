Introduction

This application is a custom developed tool for Dalibor Cmolik by Cherniaev Egor (chrnyaevek@gmail.com). Application process excel and pass each line to Chat GPT with saving results.

Setup

To setup application:

  1. Login to OpenAI account (https://platform.openai.com/) and create API key (make sure to save it somewhere!).
  2. Go to `administration` and create `Api` and `GptModel`. Check this page to learn more about models. Check `is_enabled` to activate model and api. Make sure to have only one enabled model and api!. By disabling model or api you may change model that is used in processing.

Excel uploading

To process the excel first you need to upload it to the app. The excel MUST follow this format:

  • prompt, required column with text that will be sent to Chat GPT. Each line will be filtered with HTML filter (HTML will be removed) before storing to database..
  • product_sku, optional column with prompt identifier (some unique value like product sku)
Go to home page and hit `Upload` button. In the next windows upload required excel, `tag` is dataset name for your orientation. After you press `Upload` button excel will be loaded to database.

Excel processing

After upload you may want to process excel with Chat GPT. To do so go to home page and forst of all make sure that model and api are selected in `Status` section. In the list of datasets you need to find the one you uploaded. Check that it contains expected prompts amount. To start processing hit `Process` button and confirm processing. To stop processing hit `Stop` button (no conformation). To see current status of processing refresh the page. To export results find latest `Iteration` in list of iterations on main page and hit `Export` link.

Exported file will contain modified prompt text text which was passed to model and model response. If you see error text and no response from model, that means that OpenAI server failed to process this request. Try this prompt in playground and repeat processing. Dataset is processed with all prompts that it contains! Even if prompt has already been evaluated! This simplifies system and make it more predictable, so just create new dataset with required prompts. Sometimes chat return empty message - hit retry to run over empty completions only.
